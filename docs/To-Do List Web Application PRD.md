# To-Do List Web Application

###### 1\. Objective

The objective of this project is to develop a simple, responsive, and user-friendly To-Do List web application that helps users organize and manage their daily tasks efficiently.

###### 2\. Project Scope

The application will allow users to:-

\* Add new tasks

\* Edit existing tasks

\* Delete tasks

\* Mark tasks as Completed or Pending

\* Search tasks

\* Filter tasks

\* Assign task priority

\* Set due dates

\* Store tasks permanently using a database or local storage
